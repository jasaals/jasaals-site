# Welcome to Jeff's Techy Corner

This is my little corner of the JASAALS site where I can jot down tech notes, document hobby projects, share things I'm learning, and generally geek out in writing.

## What you'll find here

- **Projects** — things I'm building or tinkering with (home lab, software, hardware)
- **Notes** — quick references and how-to's I want to remember
- **Hobbies** — anything tech-adjacent I'm into at the moment
- **Learning** — topics I'm exploring, courses I'm taking, books I'm reading

## How to add a new post

1. Add an entry to `tech-posts/index.json`:

```json
{
  "slug": "my-post-slug",
  "date": "March 2026",
  "title": "My Post Title",
  "excerpt": "A short summary shown on the listing page.",
  "tags": ["linux", "homelab"],
  "cover": ""
}
```

2. Create `tech-posts/my-post-slug.md` with your Markdown content.

That's it. The same Markdown features from the Stories section work here too — headings, bold/italic, images with captions, and even embedded videos.
